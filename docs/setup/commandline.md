# Command Line

## Install brew

1. Open terminal
2. run:

````
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
````

## Install Pyenv

1. Open terminal
2. run:

````
brew update
brew install pyenv
pyenv init
exec $SHELL
````


## Install Python
1. Open terminal
2. run:

````
brew install openssl readline sqlite3 xz zlib tcl-tk
````
